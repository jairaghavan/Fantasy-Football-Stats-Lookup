App Description:

A simple app that asks the user for an NFL player's first and last name and returns three tables. 
The first consists of the player's passing yards and touchdowns for the 2022 season, the second for rushing, and the third for receiving.


API Links:

For ESPN Player IDs: 
- https://sports.core.api.espn.com/v3/sports/football/nfl/athletes?limit=20000&active=true

To access the "lookup player" stats: 
- https://sports.core.api.espn.com/v2/sports/football/leagues/nfl/athletes/${result.id}/statisticslog
